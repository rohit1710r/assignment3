Key Features of HTML5

  1. Semantic Elements : HTML5 introduced new semantic element like <header>, <nav>,<article>,<section>,<aside> and <footer>. These elements provide a clearer structure to web page.
  2. Audio and Video Tag : HTML5 include native support for embedding audio and video content without the third party plugins. The <audio> and <video> elements allow the developers to embed media directly into web pages.
  3. Canvas : The <canvas> element provides a drawing space that allow developers to create dynamic graphics and animation using java script.
  4. Local storage : HTML5 introduced the local storage and the session storage APIs , allowing developers to store data locally on the user's device.
  5. Web Storage : In addition to local storage, HTML5 introduced the localStorage and session Storage APIs, Allowing developers to store data locally on the user's device. This features is useful for more responsive web application and reducing the need for server-side storage.
