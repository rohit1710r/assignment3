Difference between HTML and HTML5

Paramerters HTML HTML5
Defination Hyper text markup language(html) is the main language used to create a web pages. HTML5 is a updated version of HTML that bring additional features to the markup language and incorporates new Internet technology.
Multimedia Support HTML doesn't provide support for video and audio. HTML5 support both video and audio.
Internet requirement HTML require an Internet connection to access the document on the Internet. HTML5 provides user the ability to use web application to work in offline mode.
Debugging HTML code can be hard to debug as there is no error detection features. HTML5 offer better error handling and helps.
Storage HTML has flexible storage and let user store heavy files. it uses cache memory and cookies to store data. HTML5 offer several storage options and uses local storage to store data.
